NEW NEW

For the culture

New New is a web page that I created to showcase my love for music. Everyday I come across new undiscovered sounds that I feel the need to share with the world. New New allows me to share these new sounds, and present them to the world, highlighting my favourite work from their discography.

Toolset

  1. REACT JS
  2. SASS
  3. FIREBASE

The web application consisted of React JS being the core of the app, supported by SASS for styling. Firebase was integrated as the core back-end structure, providing a reliable database and back-end functionalities.

Motive

The web application was designed with a minimal approach, allowing the artist and their work to be the main focus of the website.

Discover Everywhere

New New caters to a wide variety of devices, making New New accessible while keeping it’s own feel.

The New New web application is supported by Google’s Firebase database. Firebase makes it simple to store artist information.

A back end portion of the application was also created to manage the web application’s database and to keep New New up to date with new artists.

Sharing music with others is one of the best things anyone can do. Spread the love.
