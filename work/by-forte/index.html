BY FORTE

Midterm turned passion

By Forte is a streetwear brand founded by myself and a good friend of mine. What originally started as a way to get out of a midterm, turned out to become my unexpected passion project.

Toolset

  1. REACT JS
  2. SASS
  3. FIREBASE

The web application consisted of React JS being the core of the app, supported by SASS for styling. Firebase was integrated as the core back-end structure, providing a reliable database and back-end functionalities.

Motive

Website design cues were taken from the aesthetic of the brand itself, minimal yet pleasing.

Shop On The Go

With a fully responsive design, the web application caters to all devices in various sizes, creating an easy on the go shopping experience.
